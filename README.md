# Customer-Behavior-Analysis-Dashboard-Power-BI-
This project is a Power BI dashboard created to analyze customer behavior, sales performance, and profit trends. It includes insights on regional sales, top customers, and key products. The dashboard also uses features like drill-through for detailed analysis and custom tooltips for better data understanding.
Customer Behavior Analysis Dashboard (Power BI)
Overview

This project focuses on analyzing customer behavior, sales performance, and profitability using an interactive Power BI dashboard. The dashboard provides insights into key business metrics and helps in identifying trends, top-performing regions, customers, and products.

Objectives
Analyze overall sales and profit performance
Understand customer purchasing behavior
Identify high-performing regions and product categories
Track sales trends over time for better decision-making
Key Features
Interactive dashboard with KPIs such as Total Sales, Profit, Quantity, and Profit Margin
Regional analysis (West, East, Central, South)
Customer and product-level performance insights
Time-based sales trend analysis (2021–2024)
Advanced Functionality
Drill-through feature to view detailed insights for individual customers and products
Custom tooltips to provide additional contextual information on hover
Slicers and filters for dynamic data exploration
Tools & Technologies
Power BI
DAX (Data Analysis Expressions)
Data Cleaning & Transformation
Insights Gained
Identified top-performing regions contributing the most to revenue
Recognized key customers and products driving sales
Observed sales growth trends over multiple years
Improved understanding of profit distribution and margins
Conclusion

This dashboard demonstrates how data visualization can transform raw data into actionable insights, enabling better business decisions through interactive and user-friendly reporting.
